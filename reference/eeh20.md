# Banco com 1000 registros de hospitalização do Equador.

A *Encuesta de Egresos Hospitalarios* registra as hospitalizações
ocorridas em todo o Equador, no setor público e privado. Aqui é
apresentada uma amostra aleatória da base de dados do ano 2020.

## Usage

``` r
data(eeh20)
```

## Format

An object of class `tbl_df` (inherits from `tbl`, `data.frame`) with
1000 rows and 36 columns.

## Source

<https://www.ecuadorencifras.gob.ec/>
