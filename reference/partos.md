# Identificar diagnósticos de parto na CID-10

Identificar diagnósticos de parto na CID-10

## Usage

``` r
partos(cid, opus = 1)
```

## Arguments

- cid:

  Vetor com o código diagnóstico, segundo a CID-10.

- opus:

  Ação a ser executada: 1 (padrão) seleciona apenas os partos; 2 exclui
  os partos; e 3 cria um `data.frame` com duas variáveis – `cid`, com o
  código diagnóstico, e `parto` (sim ou não).
